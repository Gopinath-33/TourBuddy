# 🌍 TourBuddy

TourBuddy is a travel booking landing page that helps users discover destinations, hotels, and travel packages across India and around the world. Built as a static front-end project showcasing domestic and international tour packages with pricing, ratings, and trip durations.

🔗 **Live Demo:** [gopinath-33.github.io/TourBuddy](https://gopinath-33.github.io/TourBuddy/)

## ✨ Features

- Hero section with search bar to discover destinations
- About section highlighting TourBuddy's mission and stats (100+ Destinations, 10,000+ Happy Travelers, 24/7 Support)
- **Domestic Destinations** — Tamil Nadu (Chennai, Ooty, Madurai, Rameswaram) & Kerala (Munnar, Alleppey, Kochi, Wayanad)
- **International Destinations** — Dubai, Singapore, Bangkok, Spain with flight, hotel & breakfast inclusions
- **Trending Packages** — Maldives Escape, Dubai Adventure, Singapore Tour
- **Traveler's Choice** — Paris, Switzerland, Bali
- Star ratings, pricing, and trip duration for each destination
- Travel category filters (College Trips, Devotional Trips, Family Tours, Friends Getaways, Office Tours, Hidden Gems, etc.)
- Reserve Trip button linking to a dedicated reservation page
- Contact section with email, phone, and WhatsApp integration
- Fully responsive design
